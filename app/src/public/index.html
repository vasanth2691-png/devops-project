Build. Ship. Observe.

A colorful DevOps playground with API health checks, container workflow, Kubernetes deployments, Helm charts, Terraform starter, GitHub Actions, and Jenkins pipeline.

Architecture

  • Developer pushes code to Git repository
  • CI pipeline runs tests and image build
  • Docker image deployed to Kubernetes
  • Service exposes API and health endpoint
  • Terraform manages cloud infrastructure

Installed Toolchain

  • Git + VS Code
  • Node.js + npm
  • Docker Desktop
  • kubectl + kind
  • Helm + Terraform
  • Jenkins + GitHub Actions

Practice Flow

  1. Run local tests
  2. Build and run with Docker Compose
  3. Deploy manifests to kind cluster
  4. Deploy same app via Helm chart
  5. Validate infrastructure with Terraform

Useful Endpoints

  • / Frontend dashboard
  • /health Live status
